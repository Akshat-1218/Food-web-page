# 🍔 Foodie - Responsive Food Landing Page

A modern and responsive food ordering landing page built using **HTML, CSS, JavaScript, and GSAP**. The project features a beautiful UI, interactive animations, a functional shopping cart, live menu search, and a responsive design for an engaging user experience.

---

## 🚀 Features

* 🎨 Modern and responsive UI
* ✨ GSAP-powered landing page animations
* 🛒 Interactive shopping cart
* ➕ Add and remove food items
* 💰 Automatic cart total calculation
* 🔍 Live menu search
* 🏷️ Filter menu items by price
* 📧 Newsletter subscription validation
* 🔔 Toast notifications
* 📱 Clean and user-friendly layout

---

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)
* GSAP Animation Library
* Remix Icons
